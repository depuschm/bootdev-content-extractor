# Privacy Policy - Boot.dev Content Extractor

**Last Updated:** January 2026

## Data Collection

Boot.dev Content Extractor does **NOT** collect, store, or transmit any personal data to external servers.

## What Data We Access

- **Boot.dev Page Content**: The extension reads content from Boot.dev pages you visit to extract challenges and lessons
- **User Settings**: Your preferences (export format, Notion settings) are stored locally in your browser using Chrome/Firefox storage APIs

## Notion Integration (Optional)

If you enable Notion integration:
- Your Notion API token is stored **locally** in your browser
- The token is **only** sent directly to Notion's API (api.notion.com) when you click "Send to Notion"
- We never see, store, or transmit your token to any other server

## Data Storage

All data is stored locally using browser storage APIs:
- Settings and preferences
- Notion API credentials (encrypted by the browser)

## Third-Party Services

- **Notion API**: If you enable Notion integration, extracted content is sent directly to your Notion workspace. This is governed by [Notion's Privacy Policy](https://www.notion.so/Privacy-Policy)

## Permissions Explanation

- **activeTab**: Read content from the current Boot.dev page
- **storage**: Save your settings locally
- **boot.dev/\***: Access Boot.dev pages for content extraction
- **api.notion.com/\***: Send data to Notion (only if you enable it)
