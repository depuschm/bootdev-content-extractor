🔗 Notion Integration
📋 Setup Instructions:
  1. Go to notion.so/my-integrations
  2. Click "+ New integration"
  3. Name it (e.g., "Boot.dev Content Extractor") and select your workspace
  4. Go to the Configuration tab and copy the Internal Integration Secret (starts with ntn_)
  5. Create one or more databases in Notion for different exercise types
  6. Add these columns to your databases for better organization:
    • Type - Property type: Select
    • Language - Property type: Select
    • URL - Property type: URL
  7. For each database: Click the "..." menu → "+ Add connections" → Select your integration
  8. Copy each database's Database ID from the URL (32-character code)
  9. Add databases below using the "+ Add Database" button
